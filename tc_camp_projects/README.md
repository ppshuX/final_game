## 🎯 项目概况

### Kotlin Calendar 日历应用

**项目周期**：2025年11月4日 - 2025年11月14日（10天）  
**当前进度**：80%（8/10天）  
**作业完成度**：✅ 100%（3个基本要求全部达成）

#### 核心功能

1. ✅ **日历视图展示**（月视图）
2. ✅ **日程增删改查**（CRUD）
3. ✅ **日程提醒功能**（AlarmManager + Notification）

#### 技术栈

- **语言**：Kotlin
- **框架**：Android SDK
- **数据库**：Room (SQLite)
- **异步**：Coroutines
- **UI**：Material Design
- **列表**：RecyclerView
- **定时**：AlarmManager
- **通知**：NotificationManager
